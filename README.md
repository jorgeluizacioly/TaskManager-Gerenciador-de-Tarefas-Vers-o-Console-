Mini projeto – TaskManager (Gerenciador de Tarefas – Console)

Este projeto foi desenvolvido como parte dos meus estudos em Ciência da Computação, com o objetivo de praticar os conceitos fundamentais da Programação Orientada a Objetos (POO) utilizando Java puro.

O TaskManager é um sistema simples de gerenciamento de tarefas que funciona no console (linha de comando). Ele permite que o usuário crie, liste, conclua e remova tarefas diretamente pelo terminal.

🎯 Objetivo

O principal objetivo deste projeto foi aplicar, na prática, os conceitos básicos da linguagem Java e os pilares da POO, criando uma base sólida para evoluções futuras, como:

Interface gráfica com JavaFX

Desenvolvimento de uma API com Spring Boot

Integração com banco de dados

🛠 Funcionalidades

O sistema permite:

✅ Criar novas tarefas

📋 Listar todas as tarefas

✔️ Marcar tarefas como concluídas

❌ Remover tarefas

A aplicação funciona totalmente em memória, utilizando ArrayList, sem persistência em banco de dados nesta versão.

📚 Conceitos Aplicados

Durante o desenvolvimento, utilizei:

Scanner para entrada de dados

System.out.println para saída no console

Estruturas condicionais (if, else, switch)

Laços de repetição (for, while, do-while)

Manipulação de listas com ArrayList<Tarefa>

Manipulação de String

Tratamento básico de exceções com try/catch

🧠 Conceitos de POO

Encapsulamento (atributos private + getters/setters)

Herança (classe TarefaPrioritaria herdando de Tarefa)

Polimorfismo

Uso de classe abstrata ou interface

Organização em pacotes seguindo boas práticas
